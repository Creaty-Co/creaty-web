# Button

- This should be a customizable button
- Should have transition on events, colors and outline type changed

## Colors

- Dark (default)
- White
- Green
- Violet

## Sizes

- Little (for `ButtonIcon`)
- Small
- Normal (default)
- Big

## Outline type

- Only border (with outline)
- Only background (without outline) (default)

## Events

- Hover
- Active
- Disabled
